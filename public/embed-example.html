How to Embed TSC Manager

You can embed the TSC Manager dashboard in your webpage using an iframe or as a Web Component.

Method 1: Using iframe

<iframe src="http://localhost:3000/?apiKey=default-api-key" width="100%" height="800px" frameborder="0" allow="web-share" ></iframe>

Method 2: Using Web Component (Advanced)

<script src="http://localhost:3000/widget.js"></script> <tsc-manager-widget api-key="default-api-key"></tsc-manager-widget>

Note: Replace default-api-key with your actual API key.
